## tiptap react

이미지 업로드 부분은 로직을 더 짜야하는데 꼬아서 짜야함.
파일업로더를 따로 만들어서 파일업로드시 해당 파일의 링크를 가져와서 tiptap 내장 함수? 에 이미지 url을 넣어줘야함.

이미지를 올리자마자 파이어베이스에 바로 저장하면 괜찮겠지만 그러면 스토리지의 낭비가 생기기 때문에 비효율적.
